Valkyrien Skies 2 is a Minecraft mod framework that adds a real physics simulation engine to the game. Players and mod developers alike can create ships, submarines, aircraft, and any other moving structure — complete with physics-based movement, buoyancy, collisions, and interactions with the world.

Whether you're a player looking to download and use VS2, or a mod developer integrating the API into your own mod, this documentation has you covered.
